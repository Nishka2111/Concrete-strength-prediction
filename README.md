# Concrete-strength-prediction
Regression-based neural network implemented in PyTorch to predict concrete compressive strength from material composition features.

# Concrete Strength Prediction using PyTorch

## 📌 Project Overview
This project builds a deep learning regression model to predict the compressive strength of concrete based on its material composition.

## 🧠 Model Architecture
- Input layer: 6 features
- Hidden layer 1: 24 neurons (ReLU)
- Hidden layer 2: 12 neurons (ReLU)
- Output layer: 1 neuron

## ⚙️ Technologies Used
- Python
- PyTorch
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Results
- Loss Function: MSELoss
- Test MSE: 111.40
